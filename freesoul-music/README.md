# FreeSoul Music · 互動音樂網頁 ✨ 13 地點全開放

## 🎬 互動方式

### 💻 桌機版（>768px）
- **角色預設隱藏**，地圖看起來乾淨整齊
- **滑鼠移到地點上** → 角色彈跳浮現 + 對應地點亮起光暈
- **點擊地點** → 彈出大圖介紹卡片
- 卡片顯示：角色立繪 + 地點名稱 + 介紹文字 + YouTube 連結

### 📱 手機版（≤768px）
- 自動切換成 **2 欄卡片網格**
- 每張卡片顯示局部背景 + 角色
- 點擊卡片彈出介紹（直接顯示，因為手機沒有 hover）

## ✏️ 補充播放清單連結

打開 `index.html`，找到 `LOCATIONS = [` 那段，每個地點都有：

```javascript
playlist: ''   ← 把網址填入引號中即可
```

填入後是這樣：

```javascript
playlist: 'https://www.youtube.com/playlist?list=PLxxxxxxxx'
```

**⚠️ 還沒填的地點**：彈窗按鈕會顯示灰色「播放清單建設中」，使用者點不到，等你填上就會自動變紅色 YouTube 按鈕。

## 🎯 微調角色位置

每個地點都有兩組座標（單位都是百分比，相對於整張地圖）：

**`hitArea`** = 滑鼠點擊熱區（看不見）
- `x, y` = 左上角位置
- `w, h` = 寬度、高度

**`character`** = 角色 PNG 顯示位置
- `x, y` = 角色「中心點」位置
- `width` = 角色寬度
- `image` = 圖檔路徑

**調整訣竅**：
- 角色太大 → 調小 `character.width`
- 角色不在島上 → 調 `character.x` 跟 `character.y`
- 點擊範圍對不到 → 調 `hitArea`
